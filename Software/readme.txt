This folder contains software associated with the Amstrad CPC Z80 DART.

drttst01.asm is a test program written in Z80 Assembly (using RASM) which transmits "Hello World".

drttst02.asm is a test program written in Z80 Assembly (using RASM) which loops back characters received.

drtlib01.asm provides libary routines and macros for the DART.

ctclib01.asm provides libary routines and macros for the CTC.
